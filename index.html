Zener Engine
A New Generation Cross-platform Java 3D Engine

Feature

Entity Component System

The main structure of Zener Game Engine is totally based on Entity Component System which is made for efficient game development.

Flexibility and Modifiability

Zener Engine is a highly customized game engine and game genre independent allowing developers to create a variety of games using various technologies.

Cross-platform

Zener Engine is completely written in notorious cross-platform language Java. It utilizes LWJGL library as a hardware interface.

Cutting Edge Graphics

Supporting the most recent OpenGL technology and GLSL script. Zener Engine also provides a real-time GLSL script parser.

Free Software

Free Software means that you have the freedom to understand the structure of the engine and contribute to the project.

Not Only 3D

Supporting Server-Client mode TCP/UDP networking by KyroNet library (in developing) and audio playback.

Functionality

  • OpenGL Shader Render System (Target for 3.3)

  • Intent and Event System
  • Input
  • Audio

  • Utility
    • Ray-cast
    • Collision

About

Zener 3D Engine is a free software created by Forrest Sun. It is built to overcome the drawbacks of traditional game engine.

Whether you're a student looking to learn 3D Game Development, a professional looking for a flexible game engine, this 3D Engine is the perfect starting point!
